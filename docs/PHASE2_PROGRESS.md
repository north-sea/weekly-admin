## 🎯 功能特性

### T2.1: 草稿管理功能

#### 卡片视图
- **响应式布局**: 根据屏幕尺寸自动调整列数
- **丰富信息**: 标题、来源、描述、标签、分类、时间
- **视觉层次**: 清晰的卡片布局，hover 效果
- **封面图**: 支持显示草稿封面图片
- **优先级**: 星星图标直观展示优先级

#### 快速预览
- **全屏对话框**: 使用 shadcn Dialog 组件
- **滚动查看**: 支持长内容滚动
- **AI 建议**: 显示分类和标签建议
- **结构化渲染**: 使用 StructuredPreview 组件

#### 批量操作
- **多选支持**: Checkbox 选择多个草稿
- **全选功能**: 一键选择/取消所有草稿
- **批量拒绝**: 批量更新草稿状态
- **选择反馈**: 已选项数量显示

#### 筛选和搜索
- **关键词搜索**: 搜索标题、描述、URL
- **阶段筛选**: 采集草稿池/编辑草稿
- **状态筛选**: 待处理/已采用/已拒绝
- **优先级筛选**: 1-5 星
- **重复项筛选**: 全部/仅原始/仅重复
- **排序选项**: 多种排序字段和方向

### T2.2: 周刊编辑器功能

#### 三栏工作台
- **左栏-内容池**: 可用内容列表，按分类折叠分组，支持搜索筛选
- **中栏-结构编辑**: 已选内容列表，支持拖拽排序和快速操作
- **右栏-实时预览**: 实时渲染周刊最终效果

#### 拖拽交互
- **从左到中拖拽**: 从内容池拖拽添加到周刊结构
- **中栏内拖拽**: 调整内容在周刊中的顺序
- **拖拽反馈**: 明确的视觉提示和放置区域高亮

#### 内容管理
- **按分类分组**: 自动按分类折叠展示内容
- **内容数量统计**: 每个分类显示包含的内容数
- **已选标记**: 已添加的内容在左栏显示为已选状态
- **快速操作**: 点击按钮快速添加或移除内容

#### 实时预览
- **周刊头部**: 显示期号、标题、描述、时间范围
- **内容统计**: 显示总内容数和分类数
- **分类展示**: 按分类分组渲染内容列表
- **精选标记**: 突出显示精选内容

## 💡 技术亮点

### 1. 完全 shadcn/ui 化
- 移除所有 Ant Design 组件依赖
- 使用 shadcn/ui + Radix UI 构建
- 遵循 claude theme 设计规范
- 统一使用 useToast 通知系统

### 2. 响应式设计
- Mobile-first 设计思路
- 网格自动适应屏幕尺寸
- 触摸友好的操作界面
- 三栏布局自适应

### 3. 性能优化
- React Query 数据管理
- 组件按需渲染
- ScrollArea 滚动优化
- 拖拽操作流畅无卡顿

### 4. 类型安全
- 完整的 TypeScript 类型定义
- Props 验证
- 数据格式验证
- Content 接口统一

### 5. 可访问性
- Radix UI 的无障碍支持
- 键盘导航
- ARIA 标签
- 屏幕阅读器友好

### 6. 用户体验优化
- **统一通知**: 使用 shadcn Toast 通知
- **加载状态**: Spinner 和禁用状态
- **错误处理**: 友好的错误提示
- **空状态**: 提示用户同步草稿或添加内容
- **操作反馈**: 即时的操作结果反馈
