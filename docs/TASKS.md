### T1.1: 安装和配置 shadcn/ui + claude theme
**状态**: 🟢 已完成  
**优先级**: 🔴 P0 (最高)  
**预估工时**: 2-4 小时  
**负责人**: TBD  

**任务描述**:
安装 shadcn/ui 及相关依赖,配置 claude theme,确保基础 UI 组件可用。

**详细步骤**:
1. 安装依赖包
   ```bash
   pnpm add @radix-ui/react-dialog @radix-ui/react-dropdown-menu \
     @radix-ui/react-slot @radix-ui/react-toast @radix-ui/react-tabs \
     @radix-ui/react-select @radix-ui/react-checkbox @radix-ui/react-switch
   
   pnpm add class-variance-authority clsx tailwind-merge lucide-react
   pnpm add tailwindcss-animate
   ```

2. 初始化 shadcn/ui
   ```bash
   npx shadcn@latest init
   ```
   选择配置:
   - Style: default
   - Base color: slate
   - CSS variables: yes
   - RSC: no (使用 client components)
   - TypeScript: yes

3. 配置 tailwind.config.ts
   - 导入 claude theme 配置
   - 设置颜色变量
   - 配置动画

4. 更新 globals.css
   - 导入 claude theme CSS 变量
   - 定义 light/dark 模式颜色
   - 添加基础样式

5. 安装基础组件
   ```bash
   npx shadcn@latest add button card input label textarea
   npx shadcn@latest add select checkbox switch toast
   npx shadcn@latest add dialog dropdown-menu tabs
   ```

6. 创建 `src/lib/utils.ts` (如不存在)
   ```typescript
   import { type ClassValue, clsx } from "clsx"
   import { twMerge } from "tailwind-merge"
   
   export function cn(...inputs: ClassValue[]) {
     return twMerge(clsx(inputs))
   }
   ```

7. 测试组件
   - 创建测试页面验证组件正常工作
   - 验证 light/dark 模式切换

**验收标准**:
- ✅ 所有 shadcn 组件成功安装
- ✅ claude theme 颜色变量生效
- ✅ 测试页面渲染正常
- ✅ 暗色模式可切换

**依赖**: 无

**详细步骤**:
1. 更新 Prisma schema
   ```prisma
   model operation_logs {
     id                Int                           @id @default(autoincrement())
     user_id           Int
     operation_type    operation_logs_operation_type
     resource_type     String                        @db.VarChar(50)
     resource_id       String?                       @db.VarChar(50) // 从 Int 改为 String
     operation_details String?                       @db.Text
     ip_address        String?                       @db.VarChar(45)
     user_agent        String?                       @db.Text
     created_at        DateTime?                     @default(now()) @db.Timestamp(0)

     user users @relation(fields: [user_id], references: [id])

     @@index([operation_type], map: "idx_operation_type")
     @@index([resource_type], map: "idx_resource_type")
     @@index([user_id], map: "idx_user_id")
     @@index([resource_id], map: "idx_resource_id") // 新增索引
   }
   ```
